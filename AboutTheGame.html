About the game

What is Subway Surfer

Subway Surfers is an endless runner where players swipe to dodge trains, jump over obstacles, and collect coins. The goal is to run as far as possible while completing missions and unlocking rewards.

How to play?

Subway Surfers is an endless running game where you control a character running along train tracks. The goal is to run as far as possible while avoiding obstacles like trains, barriers, and signs. You move the character by swiping left or right to change lanes, swiping up to jump, and swiping down to roll. Along the way, you collect coins and power-ups that help you run longer and achieve a higher score..
